# WC26 EDGE ⚽

**Private prediction hub for World Cup 2026** — player fouls, shots, corners, throw-ins, parlays and live lineup integration.

🌐 **Live site:** https://ArunSelvan2506.github.io/wc26edge

## Features
- 📅 Date picker navigation (Jun 23–28 + R32)
- 🏆 Result predictions with confidence meters
- 🟥 Player fouls — last-10 dot trackers (100%/90% hit rates)
- 🎯 Player shots — shots vs SOT separate confidence bars
- ✅ Easy bets — corners, throw-ins, goals, cards
- ⚡ Parlay builder with estimated returns
- 🤖 AI deep analysis per match (Claude claude-sonnet-4-6)
- 👕 Live lineup integration (API-Football / TheStatsAPI / Demo mode)
- ⚡ Auto odds adjustment when lineups confirmed
